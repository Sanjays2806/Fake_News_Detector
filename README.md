#**Fake News Detection using Machine Learning**
This project is an AI-based system that detects whether a news article is Fake or Real using Machine Learning and NLP.
##**Features**
  - Uses real Kaggle datasets (Fake.csv & True.csv)
  - Text preprocessing and TF-IDF vectorization
  - Logistic Regression classifier
  - High accuracy (~99%)
  - Streamlit web application for real-time prediction
